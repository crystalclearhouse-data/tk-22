# Cognitive Operations (Phase 3)

This system operates as a closed-loop cognitive organism.

## Operating Principles
- Observation precedes action.
- Action precedes escalation.
- Escalation replaces chatter.

## Autonomous Loops
- Weekly cognition runs are authoritative.
- Signal intake is signed and non-blocking.
- Duplicate signals are suppressed.

## Human Involvement
A human is notified ONLY when:
- Confidence < threshold
- Contract violation detected
- Drift exceeds tolerance

Silence indicates stability.

## Agent Constraints
- Obey SYSTEM_MAP.json
- Respect submodule boundaries
- Never guess paths or scope
- Halt on ambiguity
